# Shop Notifications!

This is a sample project to test laravel notification system.

## How it works

This repo exist to explain how to create a basic project in Laravel.

## Installation

Install this package by running cloning this repository and install like you normally install Laravel.

- Run `composer install` and `npm install`
- Run `npm run dev` to generate assets
- Copy `.env.example` to `.env` and fill your values 
- Run `php artisan key:generate`
- Run `php artisan migrate`
- Open the project in your browser, register and login to navigate.
